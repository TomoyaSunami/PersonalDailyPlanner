
# 🖼 **1. 選択日画面（週ストリップ統合）**

*“この画面だけで選択日（初期は今日）と短期が回る”*

```
 [ 週ストリップ ]  ← 今日中心に7日（選択切替／控えめ表示）
 ────────────────────────────
  Mon  Tue  Wed  Thu  Fri  Sat  Sun
<- 14   15   16   17   18   19   20　->

┌──────────────────────────────┐
│  選択日（Tue Apr 15）                             │
├──────────────────────────────┤

  [ 予定 ]  ▾アイコン（左）　　│   ← 左に3px程度の青バー
  ────────────────────────────
  09:00  チームMTG（Zoom）
  13:00  客先レビュー
  18:00  ジム
  ────────────────────────────
  ＋予定を追加

  [ タスク ] ▾アイコン（左）　  │   ← 左に3px程度の緑バー
  ────────────────────────────
  □ レポートの下書き
  □ 買い物（牛乳）
  ☑ 上司への返信（完了は薄表示・打消し線）
  （初期は3件表示、切替で全件）
  ────────────────────────────
  ＋タスクを追加

  - 週ストリップでもカレンダーでも、予定がある日は青系の丸（dot）、タスクがある日は緑系の小さな四角で表示
    - 週ストリップ: 予定あり → 青丸バッジ（badge-dot）、タスクあり → 緑の小さな四角（badge-square）
    - カレンダー: 予定あり → 青丸（.dot.event）、タスクあり → 緑の小さな四角（.dot.task）

```
