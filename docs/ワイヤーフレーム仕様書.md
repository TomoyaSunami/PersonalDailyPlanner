# 📘 ワイヤーフレーム仕様書（最新実装ベース）

## 0. 画面構造のキーポイント
- 1画面構成（`todayView`）：週ストリップと月カレンダーを同じ領域でトグル表示（週⇔月）。
- 予定／タスクリストは共通表示。日付切替は週ストリップ／カレンダー／クリックで同期。
- 予定追加ボタン＝シアングラデ、タスク追加ボタン＝グリーングラデ。
- 折り畳みトグルはシンプルなアイコン（▾／▸）、背景なし。
- 「今日」の強調：週ストリップ・カレンダーともにシアン系の枠色で統一。

---

## 1. メイン画面（選択日ビュー）
*週⇔月をトグルしながら、選択日の予定・タスクを操作する単一画面。*

```
┌─────────────────────────────────────────┐
│  日付ヘッダー         [週｜月]トグル（右上）                   │
│  例) 2025年11月26日(水)                                    │
├─────────────────────────────────────────┤

  週ストリップ（週モード） / 月カレンダー（月モード）
  ・週モード：Sun start、今日セルはシアン枠
  ・月モード：今日セルはシアン枠、予定=青ドット、タスク=緑四角
  ← prevWeek / 今週範囲 / nextWeek →   （週モード）
  ← prevMonth / 月表示 / nextMonth →    （月モード）

─────────────────────────────────────────

  ▾ 予定
  ───────────────────────────────────
  09:00  チームMTG（Zoom）
  13:00  客先レビュー
  18:00  ジム
  ＋ 予定を追加  （シアン系グラデボタン）

─────────────────────────────────────────

  ▾ タスク    [完了数表示 x/y]
  ───────────────────────────────────
  □ レポートの下書き
  □ 買い物（牛乳）
  ☑ 上司への返信（完了は薄表示＋打消し線）
  ＋ タスクを追加（グリーン系グラデボタン）
```

### インジケーター
- 週ストリップ：予定あり＝青丸ドット、タスクあり＝緑四角。
- カレンダー：予定あり＝青ドット（`.dot.event`）、タスクあり＝緑四角（`.dot.task`）。

### トグル挙動
- 右上の「週｜月」で同一領域をフェード＋スライド切替。
- トグル時にラッパー高さをアニメーションし、下のリストが押し下げられる。

### 入力と操作
- 予定追加：タイトル／日付ピッカー／任意時間／任意メモ。
- タスク追加：タイトル／任意メモ／期日（今日・明日・今週・来週・期限なし）。
- タスクは行全体クリックでチェック切替、完了は残したまま薄表示。

## 2. モーダル
### タスク追加モーダル
```
┌──────────────────────────┐
│  タスクを追加                        │
├──────────────────────────┤
  タイトル
  メモ（任意）
  期日：今日 / 明日 / 今週 / 来週 / 期限なし
  ● 作成
```

### 予定追加モーダル
```
┌──────────────────────────┐
│  予定を追加                         │
├──────────────────────────┤
  タイトル
  日付（ピッカー）
  時間（任意）
  メモ（任意）
  ● 保存
```

---

## 3. スタイル要点
- ガラスモーフィズム：半透明カード＋ぼかし＋淡い枠線。
- トグル・チップ：背景透明（トグルは黒字）、共通チップは半透明白＋枠。
- ボタン：予定=シアン系グラデ、タスク=グリーン系グラデ、ホバーで色濃く。
- 見出し余白：カレンダー/週ストリップ直下に小さな余白、その他は詰め気味。

---

## 4. ルールサマリ
- 単一ビューで週/月を切替、予定・タスクは共通リスト。
- 今日強調をシアン枠で統一。
- 折り畳みトグルは文字なしアイコン（▾/▸）、背景なし。
- タスクは完了後も表示し、件数進捗をヘッダーに表示。
